# KiroAware Demo Video Script

## Target Length: 45–60 seconds

---

### [0:00–0:05] Opening — The Problem
**Visual:** Black screen → Kiro IDE open, user typing.
**Audio:** "Kiro is powerful. But it has a blind spot."
**Text overlay:** "Kiro cannot feel your machine."

### [0:05–0:15] The Report
**Visual:** User types in Kiro chat: "My network feels really slow right now."
**Audio:** "You tell Kiro what it cannot detect."
**Visual:** Kiro calls `report_system_state` → receipt ID returned.
**Text overlay:** "User reports subjective state → Kiro captures it"

### [0:15–0:30] The Verification
**Visual:** Kiro calls `verify_user_report` → system snapshot appears.
**Visual:** Split screen: user report on left, system data on right.
**Audio:** "Kiro cross-references your experience against objective data."
**Visual:** Evidence appears: "bandwidth_drop: 80% below baseline"
**Visual:** Confidence score: 87% — VERIFIED.
**Text overlay:** "Confidence: 87% | Status: verified | Evidence: 2 items"

### [0:30–0:45] The Adaptation
**Visual:** Kiro calls `get_awareness_context` → full context panel.
**Visual:** Dashboard tabs: System Reality → User Reality → Verification → Device → Cross-Device → Kiro's Awareness
**Audio:** "Kiro now knows your machine. It adapts."
**Visual:** Adaptation card appears: "Memory is at 67%. Consider lightweight implementations."
**Text overlay:** "Kiro adapts: code complexity, suggestions, break reminders"

### [0:45–0:55] Cross-Device
**Visual:** Dashboard switches to Cross-Device tab.
**Visual:** Two devices: MacBook Pro (16GB, low complexity) vs Ryzen Workstation (32GB, high complexity)
**Audio:** "Different devices. Different rules. Same awareness."
**Text overlay:** "Cross-device learning | 2 devices | 3 patterns learned"

### [0:55–1:00] Closing
**Visual:** KiroAware logo + tagline.
**Audio:** "KiroAware. The Alien System."
**Text overlay:** "KiroAware — github.com/plastic-dude/KiroAware"
**Hashtags:** #BuildWithKiro #TeamKiro

---

## Recording Notes

- Screen capture: Kiro IDE + Terminal + Browser
- Show actual tool calls in Kiro's MCP Logs panel
- Show real system data (CPU, RAM, battery)
- Keep transitions fast — 5 seconds per scene max
- Background music: subtle, electronic, alien-themed
- End card: KiroAware logo + GitHub link + challenge hashtags
